# Lunar Outpost Optimizer

月面拠点配置最適化ゲーム - 数理最適化を体験的に学ぶ

## コンセプト

このゲームは、月面に前線基地を建設するシミュレーションを通じて、拠点配置問題における制約最適化を体験的に学習することを目的としています。

プレイヤーは以下を体験します：

- 拠点配置は「場所選び」ではなく「制約の折り合い」である
- 1点の変更が全体解を歪める
- 理論最適解と現実的解は異なる

## 主な機能

- **5種類の拠点タイプ**: 司令、発電、採掘、研究、通信
- **4つの制約条件**: 日照、地形、通信、輸送
- **動的な評価システム**: リアルタイムで制約違反を視覚的にフィードバック
- **ドラッグ&ドロップ**: 直感的な拠点配置操作

## 技術スタック

- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- Canvas API for map rendering

## 開発

```bash
# 依存関係をインストール
npm install

# 開発サーバーを起動
npm run dev

# ビルド
npm run build
```

## デプロイ

このプロジェクトはVercelでのデプロイに最適化されています。

```bash
# Vercel CLIでデプロイ
npm install -g vercel
vercel --prod
```

または、GitHubリポジトリをVercelに接続して自動デプロイを設定できます。
